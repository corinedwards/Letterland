# Keyboard Shortcuts

All shortcuts are single-key and only fire when no button or form field has focus.
Holding Ctrl, Cmd, or Alt disables them so browser shortcuts still work normally.

## Controls

| Key | Action |
|-----|--------|
| `?` | Show this shortcuts panel |
| `s` | Shuffle letter shapes |
| `d` | Toggle dark mode |
| `b` | Cycle background colour (forward) |
| `B` | Cycle background colour (backward) |
| `f` | Open free colour picker |
| `p` | Pause / resume motion |

## Letters

| Key | Action |
|-----|--------|
| `c` / `C` | Spin the C |
| `o` / `O` | Spin the O |
| `r` / `R` | Spin the R |
| `i` / `I` | Spin the I |
| `n` / `N` | Spin the N |
