Hello I’m Corin, and this is my website.

It’s an interactive doohickey that lets you mess around with and explore. It’s mostly interesting to look at, although be warned it sometimes it makes quite lurid colour combinations. But it’s also accessible and safe, so if any of this stuff is uncomfortable to your sense you have complete control to change it or to switch it all off.

Making things

I like to build things that empower people to experiment and co-create in a space that’s fun and thought provoking. So I invite you to spin about and mash up all these letters I made and drench them in which ever colours appeal to you.

You can also peeki inside the letters like an X-ray if you hit the dark mode button.

Keyboard Shortcuts

Shuffle letters = S
Dark mode = D
Background colours = B
Pause = P
Spin letters = C , O , R , I , N

About

This website uses a number of techniques to make an equality of experience for an abudance of potential users. Keybopard shortcuts allow for use with a keyboard or other assitive technology like switches. Clear HTML allows for theis website to be read out loud and to announce any changes outloud too.

There is a lot of animation here that can be annoying, or distracting, for some people, and can even be dangerous. So there is a big friendly Pause button (you can also use the Keyabord short cut P) to freeze everything in place so anyone can read in peace.
