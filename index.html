Manage Letter Shapes

Add Custom Shapes

To add custom shapes, place files in public/letters/ with this naming convention:

  • LETTER-shapename.glb - 3D model
  • LETTER-shapename.svg - SVG (auto-extruded)
  • LETTER-shapename.png - Texture (auto-matched to model)

Examples: I-diskette.glb, I-diskette.png, C-rounded.svg

Edit /public/letters/shapes-config.json to customize colors, extrusion, etc.
